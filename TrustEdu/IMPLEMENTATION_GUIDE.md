# Implementation Guide

## Quick Start

Due to PowerShell execution policy restrictions, please run commands in **Command Prompt (cmd)** or **Git Bash** instead of PowerShell.

### Install Dependencies

```bash
# Navigate to frontend directory
cd d:\Hackathon\TrustEdu\frontend

# Install lucide-react icons (run in cmd or git bash, not PowerShell)
npm install lucide-react

# Start the development server
npm start
```
